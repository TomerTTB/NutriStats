Foods Database

The Foods Database is your personal repository of food items and their nutritional information. This feature allows you to maintain a customized list of foods you commonly eat, making meal tracking quick and accurate.

Database Overview

Food Information

Each food entry contains:

  • Basic Information:
    • Food name
    • Standard portion size
    • Unit of measurement
  • Nutritional Values:
    • Calories per portion
    • Protein content
    • Carbohydrate content
    • Fat content

Managing Foods

Adding New Foods

  1. Navigate to the Foods DB page
  2. Click "Add New Food"
  3. Fill in the food details:
    • Name of the food
    • Portion size and unit
    • Nutritional information
  4. Click "Save" to add to database

Editing Foods

  • Modifying Entries:
    • Click the edit icon next to any food
    • Update the necessary information
    • Save changes
  • Deleting Entries:
    • Click the delete icon
    • Confirm deletion
    • Food will be removed from database

Using the Database

Searching Foods

  • Quick Search:
    • Type in the search bar
    • Results update as you type
    • Click a food to select it
  • Advanced Search:
    • Use filters to narrow results
    • Sort by different criteria
    • View detailed nutritional info

Food Selection

  • From Food Diary:
    • Search while adding meals
    • Quick access to common foods
    • Adjust portions as needed
  • From Database Page:
    • Browse all foods
    • View detailed information
    • Manage food entries

Best Practices

  • Keep food names clear and descriptive
  • Use consistent portion sizes
  • Double-check nutritional information
  • Regularly update commonly used foods
  • Remove unused or duplicate entries

Tips for Accuracy

  • Use reliable sources for nutritional information
  • Include brand names when relevant
  • Note any special preparation methods
  • Keep portion sizes realistic
  • Update information if products change
